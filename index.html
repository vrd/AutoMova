Just type it

AutoMova app switches keyboard layouts automatically as you type, so you can type in any* language and not be distracted by manual layout switching anymore.

* Supported layouts:
 English (United States)
 English (United Kingdom)
 English (Australia)
 English (Canada)
 Ukrainian
 Russian

Main features

  • Automatic word conversion
  • Manual conversion of last word
  • Manual conversion of selected text
  • Free and open source
